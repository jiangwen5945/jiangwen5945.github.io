## table

```css
.container {
	 display: table;
}

.left {
	display: table-cell;
  width: 30%
}
```



## flex

```css
 .container {
		display: flex;
    flex-direction: row;
}

```



## grid

```css
.container {
	 display: grid;
   grid-auto-flow: column;
}
```





## 负maigin

```css
.container {
	 overflow: hidden;
}

.left {
	margin: -99999px
	padding: 99999px
}
```
