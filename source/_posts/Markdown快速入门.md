## 二.Markdown常用语法

### 1.标题
在Markdown中，如果想将一段文字定义为标题，只需要在这段文字前面加上`#`，以此类推总共标题六级。

``` markdown
<!-- markdown语法：-->
  # 一级标题
  ## 二级标题
  ### 三级标题
  #### 四级标题
  ##### 五级标题
  ###### 六级标题
```

***

### 2.列表
1. 有序列表1
2. 有序列表2
3. 有序列表3

* 无序列表1
* 无序列表2
* 无序列表3

``` markdown
<!-- markdown语法：-->
1. 有序列表1
2. 有序列表2
3. 有序列表3

* 无序列表1
* 无序列表2
* 无序列表3
```
*注：符号与文字之间要保留一个字符的空格*

***

### 3.引用
> 如果你需要在文稿中引用一段别处的句子，那么就要用到「引用」格式。

``` markdown
<!-- markdown语法：-->

> 如果你需要在文稿中引用一段别处的句子，那么就要用到「引用」格式。
```
*tips: 引用文字与 `>` 保留一个字符的空格*

***

### 4.粗体和斜体
**我是粗体**
*我是斜体*

``` markdown
<!-- markdown语法：-->

**我是粗体**
*我是斜体*
```

***

### 5.链接与图片
``` markdown
<!-- markdown语法：-->

[显示文本](链接地址)

![显示文本](图片链接地址)
```

*tips：插入图片的语法和链接的语法很像，只是前面多了一个 ！*

***

### 6.分割线
分割线的语法只需要另起一行，连续输入三个星号 *** 即可分割两段文字内容。

``` markdown
<!-- markdown语法：-->

  ***
```

***

### 7.表格
| 左对齐标题 | 居中对齐标题 | 右对齐标题 |
| :- - - | :- - -: | - - -: |
| 短文本 | 中等文本 | 稍微长一点的文本 |
| 稍微长一点的文本 | 短文本 | 中等文本 |


``` markdown
<!-- markdown语法：-->

| 左对齐标题 | 居中对齐标题 | 右对齐标题 |
| :- - - | :- - -: | - - -: |
| 短文本 | 中等文本 | 稍微长一点的文本 |
| 稍微长一点的文本  | 短文本 | 中等文本|
```

***

### 8.代码
行级代码效果：
`<a> inline </a>`
``` markdown
<!-- markdown语法：-->
  `<a> inline </a>`
```

块级代码效果：
``` markdown
  <div>inline-block</div>
  <div>inline-block</div>
```
``` markdown
<!-- markdown语法：-->
  <div>inline-block</div>
  <div>inline-block</div>
```
