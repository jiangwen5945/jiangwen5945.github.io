## hexo基础命令

1. hexo new "文章名" == hexo n "文章名" #新建文章
2. hexo generate == hexo g    #生成本地静态文件
3. hexo server == hexo s #启动服务预览
4. hexo deploy == hexo d #部署
5. hexo clean #清除缓存

## hexo配置命令

1. npm install hexo -g #安装  
2. npm update hexo -g  #升级  
3. hexo init           #初始化
